Ubuntu 16.04.2 LTS, 64-bits
Kernel: 4.6.0-040600-generic (downgraded from 4.8.0 to build the driver)


Solution:
A.
dvb_usb_af9035 is a kernel module. 
It¡¦s incompatible to the module dvb-usb-it913x.
Please remove dvb_usb_af9035 and dvb_usb_v2 modules.
And make sure the dvb-usb-it913x, dvb-usb, dvb-core and dvb-pll modules are loaded.

B.
Intallation Steps:
1. install it913x drivers and then plug in the it913x board, you can see "dvb_usb_v2: 'ITE 9135 Generic' successfully initialized and connected".
2. sudo rmmod dvb_usb_af9035
3. sudo rmmod dvb_usb_v2
4. unplug the it913x board and plug in again
5. Now the it913x works and you can see "dvb-usb: ITEtech USB2.0 DVB-T Recevier successfully initialized and connected." in dmesg.
5. Then install it9518 driver if you like

* If use with it9518 on the same machine in the future, ALWAYS plug in it913x first then it9518
